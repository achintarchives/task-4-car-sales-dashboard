# task-4-car-sales-dashboard
Overview

This project presents an interactive sales dashboard built using Power BI.
The dashboard analyzes a car-sales dataset and provides insights into sales trends, performance by state, and breakdowns by car make and model.

Objectives

Build an interactive dashboard for business stakeholders

Identify key patterns in sales data

Use visualizations to support data-driven decision-making

Dataset

Source: Car Sales dataset (Kaggle)



Insights

Identifies top-performing states by total sales

Shows which car makes and models drive the highest revenue

Reveals month-wise or year-wise sales trends

Helps compare regional performance visually through maps

Tools Used

Power BI Desktop

Excel/CSV dataset (Kaggle)

Outcome

This project demonstrates practical dashboard-building skills, KPI selection, visual design, and interactivity setup essential for business reporting and analytics.
